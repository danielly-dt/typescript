O TypeScript (TS) é uma linguagem criada pela Microsoft que é uma extensão do JavaScript.

Isso significa que ele é JavaScript com recursos extras, principalmente a tipagem.
O TypeScript serve para:

✔️ Encontrar erros enquanto você está programando.
✔️ Deixar o código mais organizado.
✔️ Facilitar o trabalho em projetos grandes.
✔️ Melhorar o autocompletar do VS Code.
✔️ Facilitar a manutenção do código por outras pessoas.
Então o navegador entende TypeScript?

Não.

O navegador entende apenas JavaScript.

Por isso, o TypeScript precisa ser compilado (traduzido) para JavaScript.
JavaScript é como escrever uma redação diretamente.
TypeScript é escrever a redação com um professor ao seu lado corrigindo os erros antes de entregar.

O que é tipagem?

Tipagem é dizer ao computador que tipo de dado uma variável, função ou objeto pode armazenar ou receber.

Os tipos mais comuns são:

string → texto
number → números
boolean → verdadeiro ou falso
array → lista de valores
object → objetos

TypeScript é JavaScript com tipagem e outras ferramentas que ajudam a escrever código mais seguro e organizado. Ele é convertido para JavaScript antes de ser executado.
Resumindo

Tipagem é definir o tipo de dado que uma variável, parâmetro ou função pode armazenar, receber ou retornar. Isso ajuda a evitar erros e deixa o código mais seguro e fácil de entender.

É justamente por isso que o TypeScript ficou tão popular em projetos grandes: ele detecta muitos erros antes mesmo de o programa ser executado.
